Help & Support

Welcome to the Inventory Management System help center. Here are guides for core features:

How to add a new item?
Go to "Manage Items" page, fill in all required fields and click "Save" button.
How to search for items?
Use the "Search" page, enter keywords in the input box, and results will update in real time.
How to delete an item?
In "Manage Items" page, find the item and click the "Delete" button next to it.

For further assistance, please contact the system administrator.
