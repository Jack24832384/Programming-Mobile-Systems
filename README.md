#  PROG2005 Software Engineering Assignment Part1
A clean, native TypeScript-based inventory management system with full CRUD operations, search, filter, and responsive UI design.

##  Features
-  **Framework-Free**: Built with pure native TypeScript + HTML + CSS, minimal setup required
-  **Modern UI Design**: Glassmorphism style with gradient colors for a clean visual experience
-  **Fully Responsive**: Optimized for mobile, tablet, and desktop screens
-  **Strict Type Safety**: Complete TypeScript interface definitions, no implicit any, type-safe code
-  **Full Functionality**: Item management, search, popular filter, input validation, and unique ID checking

   #  PROG2005 Software Engineering Assignment Part2
An enhanced inventory management system based on Angular (Legacy Version <17), extending Part1 with multi-page routing, navigation, and a more comprehensive UI/UX design.

##  Features
-  **Multi-Page Routing**: Built with Angular Router, supporting navigation between Home and Item Management pages
-  **Upgraded UI Design**: Consistent glassmorphism + gradient style with card layouts and hover animations
-  **Fully Responsive**: Seamlessly adapts to mobile, tablet, and desktop devices
-  **Strict Type Safety**: Complete TypeScript interface definitions, maintaining type-safe code standards
-  **Extended Functionality**: All features from Part1 (CRUD, search, filter, validation) + navigation bar, page switching, and management interface
-  **Component-Based Architecture**: Modular Angular components (Home, Search, Item Manage) for better maintainability
